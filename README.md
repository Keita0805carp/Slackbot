# Slackbot

## Setup

1. Install module

```sh
pip3 install -r requirements.txt
```

2. Set TOKENs

Set TOKENs to this file.

```sh
vim keys-example.py
```

Change file name 'keys-example.py' to 'keys.py'

```sh
mv keys-exapmle.py keys.py
```

3. Run slackbot

```sh
python3 run.py
```

## Features

You can execute linux command from Slack.

```
@hoge ///(Linux command)
```

Bot reply outputs.
